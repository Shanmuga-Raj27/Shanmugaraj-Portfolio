## 🛠️ Local Development & Setup

Follow these commands to clone, configure, and boot the developer environment:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Shanmuga-Raj27/Shanmugaraj-Portfolio.git
   cd Shanmugaraj-Portfolio
   ```

2. **Install Node Packages:**
   ```bash
   npm install
   ```

3. **Start Local Hot-Reload Server:**
   ```bash
   npm run dev
   ```
